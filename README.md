Personal portfolio website for Fatema Alhosein — Full-Stack Web Developer based in Edmonton, AB.

## About

Single-page portfolio built with plain HTML, CSS, and vanilla JavaScript. No frameworks, no build tools — just clean, fast, accessible code.

Features:
- Responsive layout (mobile + desktop)
- Scroll-triggered animations
- Skills, experience, and certifications sections
- Direct contact links

## Tech

HTML · CSS (custom properties, grid, flexbox) · JavaScript (Intersection Observer API)

## Deploy

Hosted on GitHub Pages. Any push to `main` auto-deploys.
